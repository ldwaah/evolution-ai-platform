# Lesson 1 recap audio

Place the full lesson recap MP3 here. The revision player checks these filenames in order:

1. `lesson-one-recap.mp3` (preferred)
2. `lesson-one.mp3`

Expected path:

`assets/audio/citizenship/lesson-1/lesson-one-recap.mp3`

The player is wired in revision mode even if the file is missing. Add your MP3 when ready, then refresh the page.

Listening to at least 90% of the recap (or reaching the end) awards the **lesson-1-recap** emblem once per student.
